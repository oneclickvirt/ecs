# CPU性能测试排行榜

本目录包含CPU性能测试的结果和排行榜数据。

## 在线查看

可通过 https://sysbench.spiritlhl.net 访问。

## 数据字段说明

| 字段名 | 说明 |
|--------|------|
| 排名 | 在当前排序中的排名 |
| CPU型号 | 完整的CPU型号信息 |
| CPU核心数 | CPU的核心数量 |
| 单核得分 | 单核性能测试得分 |
| 多核得分 | 多核性能测试得分 |
| 多核线程数 | 多核测试使用的线程数 |
